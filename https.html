https

While http is the primary protocol used to communicate data between someones computer and the site, or more specifically the webbrowser and a website, https is the more secure version of that. Https stands for Hypertext transfer protocol secure, and what makes it more secure is that its encrypted to increase security of data transfers. So if you (or me) ever make a website that requires login credentials i would highly recommend using https over http.
